# nodemailer test

This code uses **Node** and **nodemailer** to send mails for free.

## Setup

- `npm i`
- create file in root called `.env` with this content:
```
PASSWORD = theIsThePasswordOfTheGmailAccountThatSendsEmails
```

## Execution

```
node server.js
```
